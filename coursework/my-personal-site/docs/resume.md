## Проект 2: Платформа для долгосрочной аренды недвижимости без привлечения агентов

**Описание:** Разработка платформы для долгосрочной аренды недвижимости без привлечения агентов

**Технологии:** TypeScript/JavaScript, React, Redux + Redux Toolkit, SCSS, Webpack

**Что сделано:**

- Перенёс функционал старого сайта на новый, внедрив обновлённый дизайн
- Разработал новую систему фильтрации для пользователей
- Внедрил подписную модель монетизации, благодаря которой сильно снизилась нагрузка на менеджеров компании клиента
- Подключил интерактивную карту с указанием геолокации объекта
- Адаптивная вёрстка основных страниц приложения по макетам Figma
